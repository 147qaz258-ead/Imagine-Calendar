## 任务描述

实现用户认证服务前端，包括登录/注册页面、验证码输入、Token 存储与自动刷新等。

## 技术要点

### 页面设计

1. **首页/登录页**
   - 品牌标语："看清那些没人告诉你的真相"
   - 手机号输入框（11 位格式校验）
   - 验证码输入框（6 位）
   - 获取验证码按钮（60 秒倒计时）
   - 登录/注册按钮

2. **状态管理**
   - 用户信息存储（Redux）
   - Token 存储（localStorage + Redux）
   - 登录状态管理

### 核心功能

1. **手机号校验**
   - 11 位数字格式
   - 实时校验反馈

2. **验证码发送**
   - 60 秒倒计时
   - 发送失败提示
   - 重发按钮

3. **验证码输入**
   - 6 位数字输入
   - 自动聚焦
   - 粘贴支持

4. **登录流程**
   - 新用户 → 引导页
   - 老用户 → 日历页
   - 登录失败提示

5. **Token 管理**
   - 存储 Token 到 localStorage
   - 请求拦截器自动添加 Token
   - Token 过期自动刷新
   - 401 响应自动跳转登录

### 接口调用

```typescript
// 认证服务 API
const authApi = {
  // 发送验证码
  sendCode: (phone: string) =>
    axios.post('/api/auth/send-code', { phone }),

  // 验证登录
  verify: (phone: string, code: string) =>
    axios.post('/api/auth/verify', { phone, code }),

  // 刷新 Token
  refreshToken: () =>
    axios.post('/api/auth/refresh'),

  // 登出
  logout: () =>
    axios.post('/api/auth/logout'),
};
```

### 状态机

```
初始 → 输入手机号 → 点击获取验证码 → 倒计时中 → 输入验证码 → 验证中
                                                          ↓
                                            成功 ──────→ 跳转
                                                          ↓
                                            失败 ──────→ 错误提示
```
