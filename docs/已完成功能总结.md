## 九、引导流程 (Onboarding Flow)

### 完整流程

```
用户登录成功
    ↓
[welcome] 显示产品定位弹窗
    ↓ 用户确认
[exploration] 认知边界评估 (65道问题)
    ↓ 完成评估
[preferences] 个性化偏好选择 (13维度)
    ↓ 保存偏好
[completed] 自动匹配圆桌 → 进入日历页面
```

### 状态管理

| 步骤 | 值 | 说明 |
|------|------|------|
| 欢迎阶段 | `welcome` | 显示产品定位弹窗 |
| 探索阶段 | `exploration` | 认知边界评估 |
| 偏好阶段 | `preferences` | 个性化选择 |
| 完成阶段 | `completed` | 引导完成 |

## 技术栈

### 前端
- **框架：** React 18 + TypeScript
- **状态管理：** Redux Toolkit
- **路由：** React Router v6
- **构建工具：** Vite
- **样式：** TailwindCSS

### 后端
- **框架：** NestJS
- **数据库：** PostgreSQL
- **ORM：** TypeORM
- **认证：** JWT + Passport
- **实时通信：** Socket.io
- **API 文档：** Swagger
