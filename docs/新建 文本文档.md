1. 项目介绍
1.1 一句话介绍
学生职业规划领域的“张雪峰”——帮二本三本学生看清那些没人告诉他的职业真相。我们不是帮他们找工作，是帮学生看清自己，然后自己找到路。
1.2 精简版介绍
我们的目是帮助那些二本三本、有向上发展意愿但缺乏有效信息、及背景、资源的同学。
他们最大的问题不是能力不够，而是信息太少、认识不足、身边同学对未来同样迷茫，没有多元视角。
所以我们就想：能不能把招聘信息按兴趣筛好推给他们？能不能让他们和背景相似的同学一起聊聊困惑、共同发展？于是有了这个平台：只要给对信息、帮助提升认知、提供能结识同频且互助的伙伴，这些孩子自己就能走出来。

2. 学生痛点分析
第一层：信息黑洞——不知道有什么
- 招聘信息满天飞，但不知道哪些是真的适合自己
- 没人告诉我“这个岗位到底干啥的”“这个公司到底怎么样”
- 等我知道某个机会的时候，已经截止了
第二层：认知孤岛——不知道自己要什么
- 身边同学都一样迷茫，聊不出新东西
- 不知道自己适合什么，也不知道自己差在哪
- 连“该问什么问题”都不知道
第三层：路径迷雾——不知道该怎么走
- 看到岗位要求，不知道自己该补什么、怎么补
- 简历不会写、面试没练过、没人给反馈
- 想努力但无从下手，最后只能随大流
